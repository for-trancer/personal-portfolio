Featured Project

LUNA

Luna is an advanced Android voice assistant built with Flutter that combines online and offline capabilities for seamless daily task management. Powered by a Flask server hosting XLM-RoBERTa for intent classification and NER models for online processing, Luna also features offline functionality through MobileBERT replicas trained on the MASSIVE dataset. Users can control system settings like WiFi, Bluetooth, hotspot, mobile data, brightness, and volume, while also setting alarms, launching apps, and performing YouTube searches hands-free. The assistant includes AI-driven image generation using LCM-LoRA 1.5 and offers secure access through Firebase authentication. With its modular architecture ensuring smooth performance and easy feature expansion, Luna delivers a robust digital assistant experience that works intelligently both online and offline.

NEXUS

Nexus is a modern online chatting platform developed using PHP, MySQL, and Python for secure face authentication. It enables users to engage in real-time private messaging, ensuring seamless communication across the platform. With Python-based face recognition, Nexus provides an additional layer of security for user accounts. The platform also offers customizable themes for a personalized chat experience and message translation using Google Translate, allowing effortless communication across different languages. Designed for convenience, security, and versatility, Nexus delivers a smooth and engaging chatting experience
